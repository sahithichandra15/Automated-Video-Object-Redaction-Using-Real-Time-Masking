# Automated Video Object Redaction Using Real-Time Masking

A real-time AI-powered video redaction system that automatically detects and masks sensitive objects such as faces, vehicle number plates, and other private information in videos or live webcam feeds. This project helps protect privacy by applying dynamic masking techniques using computer vision and deep learning.

---

## Features

- Real-time video processing  
- AI-based object detection  
- Automatic masking/blurring of sensitive objects  
- Supports webcam and video file input  
- Fast and efficient processing using OpenCV  
- Privacy-focused video surveillance solution

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/sahithichandra15/Automated-Video-Object-Redaction-Using-Real-Time-Masking.git
```

### 2. Navigate to the project folder

```bash
cd Automated-Video-Object-Redaction-Using-Real-Time-Masking
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```
